NLP Playground
==============

# Prerequisites

* MeCab
* NEologd
* pyenv

# Setup

```bash
pyenv install
pip install virtualenv
virtualenv venv
. venv/bin/activate
pip install -r requirement.txt
```

# Data Sets
## livedoor news corpus
https://www.rondhuit.com/download.html#ldcc

```bash
bin/datasets livedoor_news
```
